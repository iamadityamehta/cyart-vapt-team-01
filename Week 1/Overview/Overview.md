# 📌 Project Overview

This project focuses on conducting a comprehensive **Security Assessment and Vulnerability Analysis** using open-source tools. The primary objective is to identify potential security weaknesses in a system, evaluate their severity, and recommend appropriate remediation measures.

In today’s digital environment, systems are constantly exposed to threats due to misconfigurations, outdated software, and unpatched vulnerabilities. This project simulates a real-world vulnerability assessment scenario where such issues are discovered and analyzed using industry-relevant methodologies.

The assessment was carried out using **OpenVAS (Greenbone Vulnerability Manager)**, a widely used open-source vulnerability scanning tool. The tool helps in detecting known vulnerabilities, misconfigurations, and security loopholes by scanning the target system against a regularly updated database of known threats.

The project follows a structured approach:
- Setting up the vulnerability scanning environment
- Identifying the target system
- Performing vulnerability scans
- Analyzing scan results based on severity levels
- Documenting findings and suggesting remediation strategies

The vulnerabilities identified were categorized into different risk levels such as **Low, Medium, High, and Critical**, enabling prioritization of security fixes. Special attention was given to understanding the impact of each vulnerability and how it could be exploited in a real-world scenario.

This project not only demonstrates technical skills in vulnerability assessment but also highlights the importance of proactive security practices such as regular scanning, system hardening, and continuous monitoring.

Overall, this work provides a foundational understanding of how security professionals assess systems, identify risks, and implement measures to strengthen cybersecurity posture using open-source tools.
